# Plano de Expansão SEO: 3 Super Artigos (2026)

Este plano detalha a criação de 3 novos pilares de conteúdo com foco em conversão e autoridade, utilizando o SEOPack para inteligência de dados e automação de imagens com fidelidade técnica.

## 🛡️ Auditoria e Pesquisa de Mercado (Fase 1)
- **SEOPack Integration**: Acesso via `agent-browser` para extrair oportunidades de baixa concorrência e alto volume.
- **Análise Hierárquica**: Extração de H1, H2 e H3 dos top 5 concorrentes por palavra-chave.
- **Mapping de Afiliados**: Seleção de 3 produtos do nosso `affiliates.json` que melhor se encaixam nos tópicos (ex: Intelbras IFR 7000, MFR 7001, Ezviz DL05).
